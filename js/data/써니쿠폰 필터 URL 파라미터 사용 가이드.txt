# 써니쿠폰 URL 파라미터 사용 가이드

## 기본 형식
- 기본 URL 뒤에 ? 를 붙이고 파라미터를 추가합니다.
- 여러 파라미터를 사용할 때는 & 로 구분합니다.
- 동일한 카테고리에서 다중 선택 시 파라미터를 반복하여 사용합니다.

## 면세점 할인 파라미터
1. brand (브랜드)
   - 경복궁
   - 시티
   - 현대
   - 신세계
   - 부산

2. location (위치)
   - 인천1터미널
   - 인천2터미널
   - 김포공항
   - 김해공항
   - 인터넷면세점

3. product (품목)
   - 담배
   - 주류

## 여행 할인 파라미터
1. travel (여행)
   - 숙소
   - 항공권
   - 투어/티켓
   - 미식
   - 패키지

2. region (지역)
   - 일본
   - 미국
   - 베트남
   - 태국
   - 유럽
   - 필리핀
   - 대만

3. service (서비스)
   - 쇼핑
   - 교통
   - 통신
   - 보험
   - 기타

## 정렬 파라미터
- sort (정렬)
  - recommend (추천순)
  - discount (할인순)
  - popular (인기순)

## 사용 예시
1. 단일 필터 선택
```
?brand=시티
?location=인천1터미널
?product=담배
?sort=discount
```

2. 동일 카테고리 다중 선택
```
?brand=시티&brand=경복궁
?location=인천1터미널&location=인천2터미널
?region=일본&region=태국
```

3. 여러 카테고리 조합
```
// 면세점 할인
?brand=시티&location=인천1터미널&product=담배&sort=discount

// 여행 할인
?travel=숙소&region=일본&service=쇼핑&sort=popular
```

## 주의사항
- URL에 한글을 직접 입력할 경우 인코딩된 형태로 변환됩니다.
- 대소문자를 구분합니다.
- 지원하지 않는 파라미터 값은 무시됩니다.
- 파라미터 순서는 상관없습니다.
- 정렬 파라미터를 지정하지 않으면 기본값으로 추천순이 적용됩니다.

## 활용 방법
- 특정 필터 조합을 북마크하여 재사용할 수 있습니다.
- 원하는 필터 조합을 URL로 공유할 수 있습니다.
- 페이지 접속 시 자동으로 필터가 적용됩니다.
- 정렬 상태도 URL을 통해 공유할 수 있습니다.

## 정렬과 필터 혼합 사용 예시

1. 면세점 할인 페이지
```
// 할인순 + 브랜드
?sort=discount&brand=시티

// 인기순 + 위치 다중선택
?sort=popular&location=인천1터미널&location=인천2터미널

// 할인순 + 브랜드 + 위치 + 품목
?sort=discount&brand=시티&brand=경복궁&location=인천1터미널&product=담배
```

2. 여행 할인 페이지
```
// 할인순 + 여행유형
?sort=discount&travel=숙소

// 인기순 + 지역 다중선택
?sort=popular&region=일본&region=태국

// 할인순 + 여행유형 + 지역 + 서비스
?sort=discount&travel=숙소&region=일본&service=쇼핑
```

### 주의사항
- 정렬 파라미터(sort)는 한 번만 사용할 수 있습니다.
- 잘못된 정렬 예시: `?sort=discount&sort=popular` (마지막 값만 적용)
- 필터는 여러 개 선택 가능하지만, 정렬은 하나만 적용됩니다.
- 정렬 파라미터 없이 필터만 사용할 경우 기본값인 '추천순'으로 정렬됩니다.
